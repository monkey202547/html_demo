# 项目技术细节

## 1. 项目概述

本项目是一个基于 `HTML5 + CSS3 + JavaScript` 实现的响应式旅游网站示例，项目主题为“Good travel / 露比旅行站”。整体页面包括登录页、首页、注册页、特色内容页、主题介绍页，围绕课程文件 [HTML5应用开发考查要求.md](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/HTML5应用开发考查要求.md:1) 中提出的表单校验、响应式布局、多媒体控制、动画效果、样式与脚本分离等要求进行实现。

## 2. 与考查要求的对应实现

### 2.1 登录页要求

[login.html](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/login.html:1) 提供用户名和密码输入框，使用表单提交登录信息。

- 用户名通过 `maxlength="10"` 约束最大长度。
- 密码输入框使用 `type="password"` 显示密文。
- [scripts/main.js](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/scripts/main.js:1) 中的 `setupLoginForm()` 负责校验用户名长度、密码是否为 6 位数字，以及固定账号 `html5kc / 123456`。
- 登录按钮采用圆角样式，并通过 CSS 提供悬停效果。

### 2.2 首页要求

[index.html](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/index.html:1) 是登录成功后的首页，承担项目主界面展示任务。

- 页面包含日期、Logo、搜索区、导航栏和多个业务内容区。
- 左侧展示功能入口与推广链接，右侧展示内容面板、旅行助手、足迹和优惠活动。
- 布局大量采用 `grid` 和 `flex`，在 [styles/main.css](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/styles/main.css:1) 中通过媒体查询实现响应式适配。

### 2.3 注册页要求

[register.html](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/register.html:1) 对应考查中的完整 HTML5 表单设计。

- 包含用户名、密码、确认密码、性别、出生日期、兴趣爱好、邮箱、个人简介等字段。
- 使用了 `text`、`password`、`radio`、`checkbox`、`date`、`email`、`textarea` 等多种 HTML5 表单控件。
- `required`、`minlength`、`maxlength` 等原生校验与 `setupRegisterForm()` 的自定义校验配合工作。
- 重置按钮和提交按钮都使用圆角样式与悬停效果。

### 2.4 特色内容页要求

[features.html](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/features.html:1) 对应“特色内容”页面要求。

### 2.5 主题介绍页要求

[themes.html](/abs/c:/Users/罗俊翔/Desktop/2400130737/html_demo/themes.html:1) 对应“主题介绍”页面要求。

- 登录页保留了一小段内嵌样式，用于补充文档级背景效果，也符合要求中“部分样式可以采用内联或文档级样式”的描述。

## 3. 各 HTML 文件功能说明

### 3.1 login.html

功能定位：项目入口页。

- 展示品牌信息和项目简介。
- 提供登录表单。
- 调用 `main.js` 中的登录校验逻辑。
- 右侧展示卡片用于增强页面视觉与课程展示效果。

### 3.2 index.html

功能定位：项目首页，也是主要业务展示页。

- 展示旅游平台首页常见结构，如搜索、预订卡片、热门推荐、活动横幅。
- 通过多个分区模拟真实网站的信息架构。
- 作为整个项目的跳转中心，链接到注册页、特色内容页、主题介绍页。

### 3.3 register.html

功能定位：用户资料注册页。

- 使用完整表单覆盖课程要求的主要表单控件。
- 展示日期与导航，保持项目多页面统一性。
- 借助脚本提供更完整的前端校验反馈。

### 3.4 features.html

功能定位：特色内容页。

- 通过左侧菜单切换路线、美食、媒体三类内容。
- 强调“内容分区 + 右侧展示”的课程要求。
- 包含自定义视频播放控制，是 JS 交互重点页面。

### 3.5 themes.html

功能定位：主题介绍页。

- 用于将首页和特色内容中的资源，整理成“浪漫海岛、城市漫游、绿色出行”三个主题方向。
- 利用面板切换和动画，满足主题介绍页的交互与展示要求。

## 4. CSS 文件功能说明

### 4.1 styles/main.css

这是项目的主样式文件，主要实现以下内容：

- 使用 `:root` 定义全局设计变量，如主题色、阴影、圆角和背景。
- 使用 `@font-face` 提供展示型字体。
- 定义登录页、注册页、首页、特色内容页、主题介绍页的布局和卡片样式。
- 定义按钮悬停、卡片玻璃拟态、主题面板、图片卡片、统计模块等视觉效果。
- 使用两个层级的媒体查询分别处理通用页面和旅行仪表盘页面的响应式适配。
- 使用 `@keyframes` 实现浮动动画、淡入动画和进度条增长动画。

## 5. JavaScript 文件功能说明

### 5.1 scripts/main.js

这是项目统一脚本文件，按函数划分功能如下：

#### `setCurrentDate()`

- 获取页面中的日期展示元素。
- 读取当前系统日期。
- 把日期格式化后写入页面，供首页、注册页、特色内容页、主题介绍页共用。

#### `setupLoginForm()`

- 只在存在 `loginForm` 的页面执行。
- 阻止默认提交。
- 检查用户名长度。
- 检查密码是否为 6 位数字。
- 验证固定账号密码后跳转首页。

#### `setupRegisterForm()`

- 只在存在 `registerForm` 的页面执行。
- 先调用 HTML5 原生校验。
- 再检查两次密码是否一致。
- 再检查兴趣爱好是否至少选择 3 项。
- 在重置时同步清理并更新提示信息。

#### `setupFeatureTabs()`

- 获取特色内容页中的功能按钮。
- 根据 `data-feature` 切换显示对应的 `feature-view`。
- 控制按钮和内容面板的 `active` 状态。

#### `setupThemeTabs()`

- 获取主题介绍页中的主题链接。
- 阻止默认锚点跳转。
- 根据 `data-theme` 切换对应主题面板。

#### `setupMediaControls()`

- 获取视频元素、自定义按钮和音量滑块。
- 实现播放、暂停、快进、快退和音量控制。
- 在状态区输出当前控制结果，提升用户可见性。

#### `DOMContentLoaded`

- 页面 DOM 加载完成后统一初始化所有模块。
- 这种写法保证同一份脚本可以被多个页面复用。
